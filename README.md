# URPE Email Assets

Public hosting for URPE Integral Services email branding assets.
Used in HTML email templates that require publicly-accessible image URLs (Gmail, Outlook, etc.).

## Files
- `Logo-01.png` — main logo (navy background, light/gold lettering) — for dark headers
- `URPE-LEGALTECH.png` — alternate logo

## Usage
```html
<img src="https://raw.githubusercontent.com/av-ux/urpe-email-assets/main/Logo-01.png" width="115">
```
